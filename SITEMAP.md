# ARCHIVE-AI Sitemap

## 1. Authentication & Onboarding
*   Login / Sign Up (SSO with Google/Adobe)
*   Initial Workspace Setup (Import from Dropbox/Google Drive/Local)
*   AI Training Tutorial (Quick walkthrough of semantic search)

## 2. Main Dashboard (The "Hub")
*   **Activity Feed:** Recent uploads, AI suggestions, and trending tags in the workspace.
*   **Smart Drop Zone:** Persistent drag-and-drop area for new assets.
*   **Quick Stats:** Number of assets organized, time saved by AI.

## 3. The Library (Assets)
*   **All Assets:** Grid/List view with advanced sorting.
*   **Folders vs. Collections:** 
    *   *Folders:* Traditional file structure.
    *   *Smart Collections:* AI-generated groups (e.g., "Most Used," "Similar to Brand X").
*   **File Detail View:**
    *   Visual Preview (Zoom/Pan).
    *   AI Insights: Tags, Color Palette, Fonts, Text OCR.
    *   Version History.
    *   Usage Rights/License Status.

## 4. Intelligent Search
*   **Global Search Bar:** Supports Natural Language.
*   **Visual Search:** Upload an image to find similar assets.
*   **Advanced Filters:** By Format, Orientation, Color, Project, Date.

## 5. Creative Insights
*   **Brand Health:** Analysis of color/font consistency across the library.
*   **Asset Performance:** (If integrated) which assets are used most in production.

## 6. Integrations & Plugins
*   **Figma Plugin Page:** Link to download and setup instructions.
*   **Adobe Extension Hub:** PS/AI/ID plugin management.
*   **Cloud Sync:** Manage connections to Drive/Dropbox/S3.

## 7. Workspace Settings
*   **Team Management:** Roles, Permissions, Member Invites.
*   **AI Preferences:** Custom tag vocabulary, auto-processing rules.
*   **Billing & Storage:** Plan management.
